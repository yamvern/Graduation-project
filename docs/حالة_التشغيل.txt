✅ خطوات التحقق من التشغيل:
═══════════════════════════════════════════════════════════════

1. افتح المتصفح واذهب إلى:
   http://localhost:8001/api/v1/docs
   
   إذا ظهرت صفحة Swagger UI → ✅ Backend يعمل بنجاح!

2. افتح المتصفح واذهب إلى:
   http://localhost:3000
   
   إذا ظهرت صفحة Login → ✅ Dashboard يعمل بنجاح!

3. تحقق من نوافذ CMD المفتوحة:
   - يجب أن ترى رسائل "Uvicorn running on..."
   - يجب أن ترى رسائل "Ready" من Next.js

═══════════════════════════════════════════════════════════════
👤 إنشاء مستخدم Admin أولي:
═══════════════════════════════════════════════════════════════

1. افتح: http://localhost:8001/api/v1/docs

2. ابحث عن: POST /api/v1/auth/register

3. اضغط "Try it out" وأدخل:
   {
     "name": "Admin User",
     "email": "admin@watheq.com",
     "password": "admin123",
     "username": "admin"
   }

4. اضغط "Execute"

5. الآن سجل الدخول:
   POST /api/v1/auth/login
   {
     "email": "admin@watheq.com",
     "password": "admin123"
   }

6. انسخ الـ Token من الرد

7. اضغط "Authorize" في أعلى الصفحة
   وأدخل: Bearer <your_token>

   ✅ تحقق من نافذة CMD - ابحث عن أخطاء

   ✅ تأكد من تثبيت npm packages

❌ Database Connection Error:
   ✅ تأكد من أن MySQL يعمل على localhost:3306
   ✅ Database "watheq_db" سيتم إنشاؤها تلقائياً
